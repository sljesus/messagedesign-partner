# Message Design - Agencia GoHighLevel

## ¿Qué es GoHighLevel?

**GoHighLevel (GHL)** es una plataforma SaaS todo-en-uno de CRM, automatización y marketing diseñada para agencias digitales. Permite:
- Gestión de clientes y leads (CRM)
- Creación de funnels y páginas web
- Automatizaciones de mensajes (SMS, WhatsApp, Email)
- Sistema de citas y calendarios
- Gestión de reseñas de Google Business
- Chatbots y agentes de IA

**Sitio oficial:** https://www.gohighlevel.com/

---

## Estructura de GoHighLevel

### Tipos de Cuentas en GoHighLevel

| Tipo | Descripción |
|------|-------------|
| **Cuenta de Agencia** | Para gestionar clientes; puede ser "staff" (empleado) o "agency" (principal) |
| **White-label/SaaS Mode** | Revender GHL bajo tu propia marca ($497/mes) |
| **Subcuentas** | Instancias separadas para cada cliente |
| **Staff Account** | Cuenta de empleado con permisos limitados |

### ¿Qué tipo tiene Message Design?

**Message Design** cuenta con una **cuenta de agencia tipo staff** (o agency). Esta es la estructura:

```
GoHighLevel (Plataforma)
       │
       ▼
  Message Design
 (Cuenta de Agencia)
       │
  ┌────┴────┐
  ▼          ▼
Clientes   Subcuentas
(negocios   (para cada
 de México)  cliente)
```

**Nota:** No es white-label (no revendemos GHL como si fuera nuestro software). Es una cuenta de agencia desde la cual gestionamos servicios para clientes.

### Rol de SST México

**SST México** es el prestador de servicios que:
1. **Vende** los servicios a pequeños negocios en México
2. **Da soporte técnico** a los clientes de Message Design
3. **Implementa** configuraciones (reseñas, calendarios, funnels)
4. **Gestiona** la cuenta de agencia

### Problema

Los precios de Message Design ($1,497 - $8,497 MXN/mes) son **muy altos** para la mayoría de PYMEs mexicanas:

| Tipo de negocio | Ingreso mensual típico | % que podrían pagar $1,497/mes |
